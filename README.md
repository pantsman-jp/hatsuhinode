# 初日の出 2025-2026

[English](https://github.com/pantsman-jp/hatsuhinode/blob/main/docs/README.en.md)

## 概要
これは、日の出をシミュレーションするツールです。
ユーザーが指定した秒数後に日の出が始まります。

早起きして寒い思いをせずに、ブラウザ上で初日の出の雰囲気を楽しめます。

## 使い方
1. `index.html` を開く
2. 「開始ボタンを押してから何秒後に日の出が始まるか」を入力
3. 開始ボタンを押すと、指定秒後に太陽が昇り始めます
